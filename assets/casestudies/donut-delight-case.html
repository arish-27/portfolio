Donut Delight – Flavor Showcase

A vibrant and interactive product showcase prototype designed in Figma featuring delicious donut flavors.

View Interactive Prototype

The Concept

The goal was to create an eye-catching product showcase that highlights three unique donut flavors: Blueberry, Green Apple, and Caramel. Each flavor panel uses a distinct color scheme that matches the donut's personality.

The floating donut effect with scattered mint leaves creates a sense of freshness and movement, making the design feel alive and appetizing.

The Tool: Figma

This project was entirely designed and prototyped in Figma, utilizing:

  • Auto-layout for responsive panel arrangement
  • Component variants for each flavor
  • Smooth prototype transitions between screens
  • Color-coded backgrounds for visual distinction

Key Design Elements

  • Color Psychology: Purple for Blueberry (luxury, creativity), Green for Green Apple (freshness, health), Orange for Caramel (warmth, indulgence).
  • Typography: Elegant handwritten-style fonts that evoke a bakery/artisanal feel.
  • Floating Elements: The donuts and mint leaves appear to float, creating depth and visual interest.
  • Prototype Interactions: Smooth transitions between flavor panels for an engaging user experience.

Tools Used: Figma (Design + Prototyping)
