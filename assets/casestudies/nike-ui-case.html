Nike UI Web Design

A visually striking e-commerce conceptual design created using Adobe Photoshop.

The Challenge

To design a landing page for a sneaker store that breaks away from traditional e-commerce grids. Taking inspiration from the vibrant nature of street culture, the goal was to create something energetic and bold.

The Solution

I used Adobe Photoshop to craft every element of this interface. The design leans heavily on modern design trends like soft shadows, overlap effects, and bold typography ("SNEAKER").

Design Highlights

  • Dynamic Composition: The shoe is placed centrally with dynamic "floating" elements to create depth.
  • Color Psychology: A monochromatic pink/violet palette was chosen to create a distinct and memorable brand feel.
  • Typography: Large, bold sans-serif fonts are used for impact, ensuring the product name is readable even behind the product image.

Tools Used: Adobe Photoshop
