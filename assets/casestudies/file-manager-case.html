File Manager App

A minimalist and intuitive file manager designed to simplify digital organization.

Overview

The Goal was to design a clean, user-friendly file manager that avoids clutter and provides quick access to essential files. The focus was on visual hierarchy and ease of navigation.

The Solution

I utilized a card-based layout with distinct color coding for different file types (Music, Files, Videos). The "Recent Downloads" and "Memories" sections provide quick access sections that most users look for immediately. A dark mode aesthetic with vibrant accents was chosen to make the interface pop without being overwhelming.

Key Features

  • Storage Visualization: A clear, graphical representation of used vs. free space.
  • Quick Categories: One-tap access to major file types.
  • Recent Activity: Immediate visibility of recently downloaded or opened files.
